# Hero game balls

Covers for the physics balls when `?from=steam`.

## Specs

- Format: `.jpg`
- Crop: roughly **1:1**, center-weighted (shown in a circle)
- Size: **≥ 256×256** (512 is nice)

## Files in use

| File | Game |
|------|------|
| `dying-light.jpg` | Dying Light |
| `terraria.jpg` | Terraria |
| `metro-exodus.jpg` | Metro Exodus |
| `half-life-2.jpg` | Half-Life 2 |
| `counter-strike-2.jpg` | Counter-Strike 2 |
| `project-zomboid.jpg` | Project Zomboid |
| `rimworld.jpg` | RimWorld |
| `teardown.jpg` | Teardown |
| `abiotic-factor.jpg` | Abiotic Factor |
| `it-takes-two.jpg` | It Takes Two |
| `human-fall-flat.jpg` | Human: Fall Flat |
| `peak.jpg` | PEAK |

Narrow screens use the **first 9**; desktop uses all 12.
